# ڕێنمایی دامەزراندن — بەخۆڕایی

## چی دروستکراوە؟

وێبسایتێکی سادە بۆ کۆمپنیاکەت:
- زیادکردنی داواکاری کڕین
- بینینی لیستی شتەکان کە دەتەوێت بکرێن
- نیشانکردنی شتێک وەک «کڕدرا»
- بە زمانی کوردی (ڕاست بۆ چەپ)

**تێچووی ساڵانە: ٠ دینار** — هیچ سێرڤەرێکی گەورەت پێویست نییە.

---

## قۆناغ ١: هەژماری Supabase (بەخۆڕایی — بۆ هەڵگرتنی داتا)

1. بچۆ بۆ [supabase.com](https://supabase.com) و هەژمارێکی بەخۆڕایی دروست بکە
2. **New Project** بکە (ناوی پرۆژەکە بنووسە، وشەی نهێنی دابنێ)
3. چاوەڕێ بکە تا پرۆژەکە ئامادە بێت (~٢ خولەک)
4. بچۆ بۆ **SQL Editor** و ناوەڕۆکی فایلی `supabase-setup.sql` لێرە جێبەجێ بکە
5. بچۆ بۆ **Settings → API** و ئەمانە کۆپی بکە:
   - **Project URL** → لە `js/config.js` لە `supabaseUrl` بنووسە
   - **anon public key** → لە `supabaseKey` بنووسە

---

## قۆناغ ٢: ڕێکخستنی کۆمپنیا

فایلی `js/config.js` بکەرەوە و ناوی کۆمپنیاکەت بنووسە:

```js
companyName: 'ناوی کۆمپنیاکەت',
companyTagline: 'لیستی داواکاری کڕین',
```

---

## قۆناغ ٣: بڵاوکردنەوە لەسەر ئینتەرنێت (بەخۆڕایی)

### ڕێگای ئاسان: GitHub Pages

1. هەژماری [GitHub](https://github.com) دروست بکە (بەخۆڕایی)
2. Repository نوێ دروست بکە (وەک `company-website`)
3. هەموو فایلەکانی ئەم فۆڵدەرە بەرز بکەرەوە
4. بچۆ بۆ **Settings → Pages**
5. لە **Source** هەڵبژێرە: `main` branch
6. چاوەڕێ بکە ~٢ خولەک — لینکەکەت دەبێتە:
   `https://ناوی-هەژمارەکەت.github.io/company-website/`

### ڕێگای تر (هەمان بەخۆڕایی):
- [Netlify](https://netlify.com) — فۆڵدەرەکە ڕاکێشە و بیخەرە سەر
- [Cloudflare Pages](https://pages.cloudflare.com)

---

## تاقیکردنەوە لە کۆمپیوتەرەکەت

فایلی `index.html` ڕاستەوخۆ لە وێبگەڕ بکەرەوە، یان:

```bash
npx serve .
```

---

## پرسیارە باوەکان

**ئایا پارە دەدەم؟** نەخێر. GitHub Pages + Supabase هەردووکیان پلانی بەخۆڕاییان هەیە کە بۆ کۆمپنیاکەت بەسە.

**ئایا سێرڤەر پێویستە؟** نەخێر. هەموو شت لەسەر خزمەتگوزاری بەخۆڕایی دەڕوات.

**کێ دەتوانێت داواکاری زیاد بکات؟** هەر کەسێک لینکەکەی هەبێت. ئەگەر تەنها بۆ تیمەکەت بێت، لینکەکە بە نهێنی بهێڵەرەوە.
